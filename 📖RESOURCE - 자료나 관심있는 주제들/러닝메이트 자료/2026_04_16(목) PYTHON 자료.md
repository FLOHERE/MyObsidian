4. 문제 함수

```python
# 1. 자판기 함수 만들기 (기계 설계도)
def vending_machine(money, choice):
    # 만약 돈을 1000원 이상 넣었다면?
    if money >= 1000:
        result = choice + " 나왔습니다! 캔을 꺼내주세요."
        return result
    # 돈이 1000원보다 적다면?
    else:
        return "돈이 부족합니다. 다시 돈을 넣어주세요."

# 2. 자판기 사용해보기 (함수 호출)
# 상황 A: 1500원을 넣고 콜라를 골랐을 때
my_drink1 = vending_machine(1500, "콜라")
print("첫 번째 결과:", my_drink1)

# 상황 B: 500원만 넣고 사이다를 골랐을 때
my_drink2 = vending_machine(500, "사이다")
print("두 번째 결과:", my_drink2)
```

5. print와 return 

- `print` : 단지 값을 화면에 보여주는 형태
- `return` : 함수가 일한 결과값을 프로그램(나)에게 돌려주는 역할을 한다. -> 그래서 우리가 그 값을 변수에 저장해서 또 써먹을 수 있다!
