3. **. (조건문과 반복문 / 코드작성형)** 1부터 10까지의 정수 중에서 **'짝수'**만 출력하는 프로그램을 작성하려고 합니다. `for`문과 `if`문, 그리고 나머지 연산자(`%`)를 모두 사용하여 코드를 완성하시오.

```python

    if i % 2 == 0:
        print(i)
```
