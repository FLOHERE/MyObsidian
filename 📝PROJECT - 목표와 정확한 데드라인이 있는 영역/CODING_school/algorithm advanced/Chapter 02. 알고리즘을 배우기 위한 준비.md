## 2.1 알고리즘이란?
- 문제를 해결하는 단계적 절차 또는 방법
	- 여기서 주어지는 문제는 컴퓨터로 푼다.

- 알고리즘의 일반적인 특성
	1. 정확성 : 주어진 입력에 대해 올바른 해를 줘야 한다.
	2. 수행성 : 각 단계는 컴퓨터에서 수행이 가능해야 한다.
		- 알고리즘 애매 모호 => 프로그래밍 언어로 변환 불가능
	3. 유한성 : 유한 시간내에 종료되어야 한다.
	4. 효율성 : 효율적일수록 가치가 높다.
		- 알고리즘은 항상 시간, 공간적 효율성을 갖도록 고안되어야 한다.
		- 입력의 크기가 커질수록 가치를 더함.
