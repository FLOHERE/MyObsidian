## 1. 정렬 알고리즘이란?
- 알고리즘 데이터를 일정한 순서대로재배열 하는 알고리즘
	- 오름차순 or 내림차순으로 재배치
		- 오름차순 : 작은수 -> 큰수
		- 내림차순 : 큰수 -> 작은수

## 2. 선택정렬
- 데이터를 일정한 순서대로 재배열 하는 정렬 알고리즘
- 동작 방식 : 가장 작은 값을 찾아 현재 위치의 값과 자리를 바꾸는 형식(이것 반복)
- 장점
	1. 구현이 간단하다.
	2. 메모리를 거의 사용하지 않는다.(오버헤드, 임베디드 등 메모리가 제약저깅 환경에서 유리하다)
	3. 로직이 직관적이다.
- 특징
	- 교환 횟수가 적다.(회전당 한번)
	- 데이터 크기가 작을수록 유리하다.
- 예시 : 키,나이,점수 등 낮은 값 -> 높은값 나열


>[!question] 배열 : {9,7,8,2,5} 를 정렬 하세요
>

```java
for(int i = 0; i< n-1; i++){
	int minIndex = i;
	for(int j = i+1; j<n; j++){
		if(arr[j] < arr[minIndex]){
			minIndex = j;
		}
	}
	swap(arr, i, minIndex); // 여기서 교환
}
```
