- 유클리드가 뭘까?
	- 두 수 a와 b(a>b)가 있을때 두 수의 최대 공약수는 a를 b로 나눈 나머지 r과 b의 최대 공약수와 같다.
- 표현 : $GCD(a,b) = GCD(b, mod b)$
	- ex_)48, 18의 최대 공약수는?
		=> 48/18 = 2...12 (r = 12)
		=> 18/12 = 1...6 (r = 6)
		=> 12/6 = 2...0 (r = 0)
		- $mode = 6$

	- 데이터가 정렬되지 않아도 사용 가능

	- 직관적, 코드 단순, 빠르게 구현 가능
