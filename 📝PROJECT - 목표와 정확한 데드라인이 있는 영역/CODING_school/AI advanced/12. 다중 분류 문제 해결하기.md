## 3. 원-핫 인코딩
- 원-핫 인코딩 : 여러개의 문자열 값을 0과 1로 이루어지도록 하게 만든다. 

```python
X = df.iloc[:0:4]
y = df.iloc[:,4]

print(X[0:5])
print(y[0:5])
```

![[Pasted image 20250619174331.png]]

- y값을 보면 숫자가 아니라 문자로 나온다.
	- 딥러닝은 계산을 위해 문자를 모두 숫자형으로 바꿔주어야 하므로 "원-핫 인코딩"을 사용하여 문자열을 숫자로 변환한다.
- 여기서는 자신의 이름이 일치하는 경우 1, 나머지는 0으로 바꾸도록 함

![[Pasted image 20250619174502.png]]
