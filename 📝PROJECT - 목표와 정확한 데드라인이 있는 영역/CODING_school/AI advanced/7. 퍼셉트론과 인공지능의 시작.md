# 1. 인공지능을 알린 퍼셉트론
## 1. 뉴런과 시냅스
- 인간의 뇌 신경세포에서 영감
	1. 뉴런 자극
	2. 화학 물질 분비
	3. 전위가 일정 값에 도달
	4. 시냅스를 통해 다른 뉴런으로 신호 전달
## 2. 퍼셉트론
- 인공신경망을 실체화 한 장치
	- 여러 입력값을 받아 각각에 가중치를 곱하여 합산.
		- 이 합산된 값이 특정 임계치를 넘으면 1 출력, 아니면 0 출력
	- 입력값에 가중치 조절 -> 학습을 통해 다양한 문제 해결 가능
		- 학습 : 알고리즘이 데이터를 바탕으로 규칙을 찾아내는 과정
