### 2. 다층 퍼셉트론 설계
1. 각 입력값  * 가중치 + 바이어스
2. 은닉층으로 전송
	=> 노드 : 이 값들이 모이는 은닉층의 중간 정거장(n1,n2)
3. 각 노드 값 계산(은닉층에서 취합)
4. 각 노드 값들은 활성화 함수(시그모이드 등) 통과
5. 예측값 출력
