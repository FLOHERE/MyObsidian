## 1. 모델의 정의 
- 딥러닝이 실제로 수행되는 부분

```python
model() : 딥러닝 모델 설정 및 구동
Sequantial() : 딥러닝 구조 설계 및 층 설명
model.compile() : 모델 컴파일
model.fit() : 모델 실제 수행 부분(모델 학습 실행)
```

## 2. 입력층, 은닉층, 출력층
- 딥러닝 : 입력층과 출력층 사이의 은닉층들을 차곡차곡 추가하여 학습 시키는 것.
	- 전반적인 흐름
		1. 입력층에서 데이터 입력함
		2. 은닉층에서 입력값과 가중치를 곱함
		3. 활성화 함수를 통과
		4. 결과값 출력
- 은닉층은 `sequantial()` 함수로 생성
### 1. Dense 함수
- 각각의 층은 `Dense` 라는 함수를 통해 구조가 결정됨(층을 만들기 위한 구성 요소)
- Dense 함수의 구성 요소
	- `model.add()` : 새로운 층 생성
		- 맨 마지막 층은 출력층이다.

>[!note] 노드 = 가중합
>이전 층에서 전달된 변수와 가중치, 바이어스가 하나로 모이게 되는 곳

```python
model.add(Dense(30, input_dim = 16))
```

1. 첫번째 Dense 층 : 입력 및 은닉 층 역할
	- 첫번째 인자에 몇개의 노드를 만들것인지 정함
	- `30` : 노드의 개수
	- `input_dim` : 전체 데이터 중 몇개를 추출 할것인지 정하는 함수
		- `16` : 16개의 데이터를 입력 하겠다는 뜻

### 2. 출력층(활성화 함수)

```python
model.add(Dense(1,activiation = 'sigmoid'))
```

2. 마지막 층 = 출력층
	- 값이 하나가 나와야 함.
	- `1` : 노드 수
	- `activation` : 활성화 함수를 어느것으로 지정 할지
	- `sigmoid`(시그모이드 함수)  : 0 or 1로 값이 나옴

## 3. 모델 컴파일
- `model.compile()`모델 컴파일 : 모델을 효과적으로 사용하기 위해 컴파일(여러 환경 설정)

### 1. 손실 함수 정하기
1. 선형회귀 : 평균 제곱 오차
2. 로지스틱 회귀 : 교차 엔트로피 오차
	- 0,1로 답이 나와야 할때 사용
	- `binary_crossentropy`
3. 다항 분류
	- `categorical_crossentropy`

```python
model.compile(loss = 'binary_crossentropy', optimizer = 'adam', metric = ['accuracy'])
```

- `loss` : 손실함수 
	- ` = 'binary_crossentropy'`:  교차 엔트로피 로 지정한다.
- `optimizer` : 가중치 설정
	- ` = 'adam'` : 아담으로 설정
- `metrics` : 모델 수행 결과
	- ` = 'accuracy'` : 학습셋 기반 정확도 출력

## 4. 모델 실행하기
1. 모델 정의 
2. 모델 컴파일
3. 모델 실행

```python
model.fit(X, y, epochs = 5, batch_size = 16)
```

- `model.fit()` : 컴파일 단계에서 정해진 환경을 주어진 데이터를 불러 실행시킬때 사용하는 함수
	- `epoch` : 모든 샘플에 대해 한번 실행 되는것
		- = 5 : 다섯번 반복하겠다.
	- `batch_size` : 샘플을 한번에 몇개씩 처리할지 정함
		-  = 16 : 470개의 샘플을 16개씩 끊어서 집어 넣어라.
		- batch_size는 크기가 크면 속도가 느려지지만 정확도는 올라간다.
		  반면, 크기가 작으면 속도는 빨라지지만 정확도는 떨어진다.
		  -> 본인 컴퓨터에 맞게 메모리 보고 알잘딱으로 정하세요!

>[!note] 용어 복습
> 속성 : 각 정보
> 라벨 : 정답지
> 샘플 : 입력할 데이터의 개수
